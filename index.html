O que é sistema operacional?

sistema operacional e um software que converte os "sinais" que voce passa ao computador à uma linguagem de baixo nivel chamada linguagem binaria que o computador entenda.

UNIX

foi criado devido a falha do sistema multics e é a base de quase todos os sistemas operacionais. Criado na linguagem massembly por ken thompson, que queria continuar jogando um jogo em um SO atual e que rodasse em um computador antigo.

macOS

um dos primeiros sistemas operacionais com interface grafica, exclusivo da apple, chamado inicialmente de apple macintosh system, lançado em 1984.

windows

um sistema que originalmente era uma interface gráfica que rodava sobre o MS-DOS, um sistema atualmente muito conhecido, tendo várias versões como 1.0, 2.0, 3.0, 95, 98, 98 SE, e com muitas ferramentas próprias, desenvolvido pela microsoft.

linux

um sistema operacional com várias "versões" chamadas distrus, que sao vários sistemas com a mesma base e com suas funções. Eles são desenvolvidos pela comunidade o tornando open source.
